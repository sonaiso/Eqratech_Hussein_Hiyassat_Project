# القياس — طبقات القياس والتقييم

لا تكتمل الخطة دون جعل **القياس** مسارًا صريحًا. القياس هنا يعني:

* **قياس دقة الجذر** — مقارنة مخرجات L8 بمرجع ذهبي (مثل Mishkat)، وحساب accuracy/precision/recall للجذر.
* **قياس دقة الوزن** — تقييم L9 (الوزن الصرفي) مقابل معيار معروف.
* **قياس دقة الإعراب** — تقييم L11 مقابل إعراب مرجعي (مثل quran_i3rab أو عينات يدوية).
* **قياس دقة تصنيف الجملة** — تقييم L12 (نوع الجملة، الأساليب) مقابل ذهبي.
* **قياس الأداء** — زمن التنفيذ ووقت كل مرحلة (موجود حاليًا في Profiling؛ يُستكمل بمعايير وحدود).
* **تحليل الأخطاء** — تصنيف نوع الخطأ (جذر، وزن، إعراب، صرف، نحوي، بلاغي) وليس فقط نسبة الدقة.

هذه الطبقات تُترجم لاحقًا إلى إطار تقييم ذهبي (Stage 22) ومحرك تصنيف الأخطاء (Stage 23)، وتُغذّي التقارير العلمية (Stage 24) واستراتيجية التحسين (Stage 25).
