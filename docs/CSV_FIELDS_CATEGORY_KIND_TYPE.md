# الفرق بين الحقول: category، kind، type

في CSV المورفولوجيا (مثل `out_with_sources.csv` و`out_with_resolver-old.csv`):

## kind (النوع العريض للكلمة)
- **المصدر**: مصنّف الكلمات `WordClassifier` (طبقة c2b).
- **المعنى**: التصنيف الرئيسي للكلمة: هل هي اسم، فعل، أداة، حرف، ضمير، إلخ.
- **قيم محتملة**: `noun`, `verb`, `operator`, `particle`, `pronoun`, `name`, `mabni`, `unknown`, `demonstrative`.
- **عربي**: اسم، فعل، أداة، حرف، ضمير، اسم علم، مبني، غير معروف، إشارة.

## category (الصنف الصرفي)
- **المصدر**: من الوزن/النمط (pattern) أو من السمات عند الاشتقاق.
- **المعنى**: صنف الصيغة من حيث الاسمية/الفعلية: عادة "اسم" أو "فعل" عندما تكون الكلمة مشتقة ولها وزن معروف (مصدر، اسم فاعل، صيغة فعل، إلخ).
- **قيم محتملة**: غالباً `noun` أو `verb` أو فارغ عندما لا يُحدد الوزن.
- **عربي**: اسم، فعل.

## type (نوع الجذر أو نوع الوزن)
- **المصدر**: إما من **نوع الجذر** (root_type): ثلاثي/رباعي، أو من **نوع الوزن** (pattern_type): صيغة الفعل (form_i, form_ii, …)، أو اسم فاعل، جمع، إلخ.
- **المعنى**: تفصيل إضافي: هل الجذر ثلاثي أم رباعي؟ أو أي صيغة للوزن (ماضي، مضارع، اسم فاعل، جمع تكسير، …).
- **قيم محتملة**: `trilateral`, `quadrilateral`, `form_i`, `form_ii`, `active_participle`, `unknown`, إلخ.
- **عربي**: ثلاثي، رباعي، صيغة أولى، صيغة ثانية، اسم فاعل، غير معروف، إلخ.

## مثال من الصف
```
root_type   category   kind   type
trilateral  noun       noun   unknown
```
- **root_type**: الجذر ثلاثي (ثلاثي).
- **category**: الصنف الصرفي للوزن = اسم.
- **kind**: تصنيف الكلمة = اسم.
- **type**: نوع الوزن غير محدد = غير معروف.

استخدام `--arabic-tags` في CLI يكتب هذه القيم (وقيم حقول أخرى مثل case, number, gender) بالعربية في CSV.
