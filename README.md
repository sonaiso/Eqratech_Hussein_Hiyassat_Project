# Eqratech_Arabic_Diana_Project
Python_NLP Project with all Arabic tools verbs and names

## Projects

### Main Project - Arabic Grammar Analysis
The core project contains various engines for Arabic linguistic analysis including:
- Phonemes and Harakat engines
- Verb and noun processing
- Sentence generation and analysis
- Grammar reconstruction utilities

### SFGCOQ - Arabic Formal Verification
A new subproject for formal verification of Arabic grammar using:
- **Systemic Functional Grammar (SFG)** - Linguistic theory by M.A.K. Halliday
- **COQ** - Formal proof assistant for mathematical verification
